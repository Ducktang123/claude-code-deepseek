【这是什么】
  让 Claude Code（命令行 AI 编程助手）使用 DeepSeek 模型的一键安装包。
  全程使用国内镜像（npmmirror / 清华 PyPI），国内网络可直接安装，无需梯子。

【安装步骤】
  1. 把整个文件夹解压到任意位置（路径最好不含特殊字符）。
  2. 右键【安装.bat】→ 以管理员身份运行（推荐，便于自动装 Node/Python）。
     （也可直接双击，多数情况也能装。）
  3. 按提示操作：
       - 自动检测/安装 Node.js
       - 自动安装 Claude Code
       - 输入你的 DeepSeek API Key（去 https://platform.deepseek.com 创建）
       - 自动配置环境变量、安装技能
  4. 看到"安装完成"后，关闭所有终端窗口。

【开始使用】
  1. 新开一个终端（PowerShell 或 CMD）。
  2. 进入任意项目文件夹，输入：  claude
  3. 在 claude 里输入  /  可以看到所有技能，输入 /名称 调用。

【已包含的技能】
  编程类 : code-review(代码审查) debug(调试) explain-code(讲解代码)
           write-tests(写测试) git-commit(生成提交信息)
  文档类 : docx(Word) xlsx(Excel) pptx(PPT) pdf(PDF)
  写作类 : tech-writing(技术文档) markdown-format(Markdown 排版)
  中文类 : translate-zh(中英翻译)

  注：文档类技能需要本机有 Python（安装时会尝试自动安装）。

【技能怎么用】（两种方式，任选）
  方式1：在 claude 里输入  /  会弹出技能列表，选一个，或直接打 /技能名
  方式2：不用记名字，直接用大白话说需求，claude 会自动挑合适的技能
         例：把这个文件夹的代码审查一下 / 帮我把这份 Word 提取成文字

【技能详细说明】（12 个）

  —— 编程类 ——
  /code-review  代码审查
     作用：找出代码里的 bug、安全隐患、可简化处。
     例：  /code-review   或   "审查一下我刚改的代码"

  /debug  调试排错
     作用：贴报错给它，系统化定位根因并修复。
     例：  把报错信息贴进去，说 "帮我看看这个错怎么解决"

  /explain-code  讲解代码
     作用：用通俗中文讲清一段代码/函数的作用和流程。
     例：  /explain-code   然后指定文件，或 "讲讲 main.py 这个函数"

  /write-tests  写单元测试
     作用：为指定函数/文件生成测试并跑通。
     例：  "给 utils.py 里的函数写单元测试"

  /git-commit  生成提交信息
     作用：看你的改动，生成规范的 git commit 信息（确认后才提交）。
     例：  改完代码后输入 /git-commit

  —— 文档类（需要 Python）——
  /docx  Word 文档
     作用：读取/生成/修改 .docx，提取文字、做报告、批量替换。
     例：  "把这份 Word 里的文字全部提取出来"

  /xlsx  Excel 表格
     作用：读写 .xlsx，汇总统计、生成报表、写公式。
     例：  "统计这个 Excel 里每个类别的金额总和"

  /pptx  PPT 演示
     作用：读取/批量生成幻灯片。
     例：  "根据这份大纲帮我生成一个 PPT"

  /pdf  PDF 处理
     作用：提取文字/表格、合并拆分、提取指定页。
     例：  "把这个 PDF 的前三页单独存成一个文件"

  —— 写作类 ——
  /tech-writing  技术文档
     作用：写/改 README、说明书、API 文档、教程。
     例：  "给这个项目写一份 README"

  /markdown-format  Markdown 排版
     作用：把杂乱文本整理成规范、好看的 Markdown。
     例：  "把这段文字整理成 markdown 格式"

  —— 中文类 ——
  /translate-zh  中英翻译
     作用：中英互译并润色，技术内容尤其准。
     例：  "把这段英文翻译成中文"   /   "这段话翻成英文"

  小提示：文件类操作（Word/Excel/PDF 等），把文件放进你用 claude 的
          那个文件夹，再让它处理，它就能找到。

【使用的模型】
  主力模型 : deepseek-v4-pro    （DeepSeek V4 最新，强推理，适合编码）
  快速模型 : deepseek-v4-flash  （内部小任务用）

【常见问题】
  Q: 输入 claude 提示找不到命令？
  A: 没重开终端。关闭所有窗口重新开一个；仍不行就重新运行【安装.bat】。

  Q: 想换 Key 或换模型？
  A: 重新运行【安装.bat】，重新输入即可。

  Q: 文档技能用不了？
  A: 需要 Python。装好 Python 后重新运行【安装.bat】会自动装依赖库。

  Q: 提示安全/权限拦截？
  A: 用管理员身份运行【安装.bat】。

【常见报错速查】（按现象对号入座）

  ① 窗口卡住不动，标题栏开头出现"选择"两个字
     原因：鼠标点了黑窗口，进入"选择/标记"模式，程序被暂停。
     解决：在窗口里按一下 ESC（或回车），程序立刻继续。
     根治：窗口左上角图标 → 属性 → 取消勾选"快速编辑模式"。
     ★ 安装过程中不要用鼠标去点窗口里的文字。

  ② npm 一直卡住，或报错里有 127.0.0.1:10868 / proxy / ECONNREFUSED
     原因：开着加速器 / VPN，代理拦截了 npm。npmmirror 是国内源，应直连。
     解决：先关掉加速器 / VPN，再重新运行【安装.bat】。
           （脚本会自动临时清代理，但代理设在系统级时仍建议手动关掉。）

  ③ claude 里发消息报 "API Error: 400"
     原因：DeepSeek API Key 没填对（最常见：粘贴时多了空格，或没粘上）。
     自查：新开 PowerShell，运行下面一行看 key 长度（正常 35 位）：
           [Environment]::GetEnvironmentVariable("ANTHROPIC_AUTH_TOKEN","User").Length
     解决：重新设置（.Trim() 会自动去掉多余空格，把引号里换成你的 key）：
           $k = "你的key".Trim()
           [Environment]::SetEnvironmentVariable("ANTHROPIC_AUTH_TOKEN",$k,"User")
     然后关掉所有终端、重新开一个再用 claude。

  ④ claude 报 401 / Authentication Fails
     原因：key 错、失效，或账户没充值（DeepSeek 无免费额度）。
     解决：去 platform.deepseek.com 确认 key 正确、账户已充值。

  ⑤ 改了 key / 环境变量后不生效
     原因：环境变量只对"新开的"终端生效。
     解决：关闭所有终端窗口，重新开一个，再运行 claude。

  ★ 黄金法则：改完任何配置，一律"关掉所有终端 → 重新开一个"再测。
