# 中英翻译与润色

做准确、地道的中英互译，技术内容尤其要专业。

## 步骤
1. 判断方向（中→英 或 英→中）和文体（口语/正式/技术文档）。
2. 翻译时：
   - **技术术语**保持准确，常见术语用业界通用译法，必要时保留英文原词（如 API、token、commit）。
   - **代码、命令、变量名、文件路径不翻译**，原样保留。
   - 通顺地道，符合目标语言表达习惯，不要逐字硬翻。
3. 如果原文有歧义，给出最可能的译法并说明。
4. 较长内容可中英对照排版，方便核对。

## 原则
- 准确第一，通顺第二，优雅第三。
- 不擅自增删原意；不确定的专有名词保留原文。
- 报错信息翻译时，保留关键英文关键字（方便用户搜索）。
