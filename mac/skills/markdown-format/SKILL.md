# Markdown 排版整理

把内容整理成干净、规范、好读的 Markdown。

## 步骤
1. 理解原文结构（哪些是标题、列表、代码、表格、引用）。
2. 套用规范：
   - **标题**：层级从 `#` 起逐级递进，不跳级。
   - **列表**：无序用 `-`，有序用 `1.`；嵌套缩进两个空格。
   - **代码**：行内用 `` ` ``，多行用 ```` ``` ```` 并标注语言。
   - **表格**：用 `|` 对齐，加表头分隔行。
   - **强调**：重点用 `**加粗**`，少用斜体。
   - **链接/图片**：`[文字](URL)` / `![描述](URL)`。
3. 统一中英文之间的空格、标点，去掉多余空行。
4. 输出整理后的完整 Markdown。

## 原则
- 只改格式，不改原意；内容有歧义先问。
- 结构清晰优先，不过度装饰。
- 中文文档保留中文标点，代码/命令保持原样。
